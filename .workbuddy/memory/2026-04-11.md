### FastAPI + pywebview 版文件结构
```
li_jie_hr/
├── main.py              ← 启动 FastAPI 后台线程 + pywebview 窗口（http://127.0.0.1:8765）
├── api_server.py        ← FastAPI 所有路由（CORS 已配置）
├── services/
│   ├── db.py            ← SQLite 连接 + 初始化
│   └── crud.py          ← 所有数据库 CRUD 函数
├── database/
│   ├── db_manager.py    ← 保留（兼容）
│   └── schema.sql       ← 数据库结构（不变）
├── web/
│   └── index.html       ← 前端 SPA，JS 用 fetch() 调用 FastAPI
├── run.bat              ← 一键启动（自动安装依赖 + 启动）
├── data.db              ← SQLite 数据库
└── requirements.txt    ← fastapi==0.111.0 / uvicorn[standard]==0.29.0 / pywebview==5.1 / python-multipart==0.0.9
```

### 架构说明
- FastAPI 运行于 127.0.0.1:8765，提供所有 HTTP API
- pywebview 加载 http://127.0.0.1:8765
- 前端 JS 用 fetch() 同源调用 FastAPI（CORS 已允许所有来源）
- FastAPI 在 daemon 线程中启动，不阻塞主线程

### 已实现功能（FastAPI + pywebview 版）
1. 成员管理：卡片视图 + 月份选择器 + 工资明细 + 人工增扣弹窗
2. 部门管理：左侧树形导航 + 右侧详情，大部门/小部门增删
3. 型号单价表：动态列表格，行=型号，列=小部门，实时保存
4. 做货编辑：**Excel 风格可编辑表格**，订单号/型号下拉，Tab 键跳转，直接填入每人每型号对数
5. 总工资表：按大部门分组，含部门小计和全厂合计

### 数据库默认数据

- 小部门：衣车、折面、折边（属面部）；底部（属底部）
